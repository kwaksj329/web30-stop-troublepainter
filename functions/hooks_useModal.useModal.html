Function useModal

Modal을 열고 닫는 기능을 제공하는 커스텀 훅입니다. 모달이 열릴 때 autoCloseDelay가 설정된 경우 자동으로 모달을 닫을 수 있습니다. 'Escape' 키를 누르면 모달을 닫는 기능이 포함되어 있습니다.

Example

const { openModal, closeModal, handleKeyDown, isModalOpened } = useModal(5000);

// 모달 열기
openModal();

// 모달 닫기
closeModal();
